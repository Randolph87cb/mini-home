# mini-home

首版目标是做一个“双人共享线上小家”的网页小客厅。当前仓库已经包含首版场景素材、场景清单和一版静态前端实现。

## 当前文件

- `index.html`：首版页面入口。
- `styles.css`：首版页面样式和场景布局样式。
- `app.js`：首版场景渲染、互动逻辑和拖拽逻辑。
- `assets/scene-manifest-v1.json`：首版场景资源清单和初始布局数据。

## 运行方式

不要直接双击 `index.html` 打开。当前实现通过 `fetch` 读取 `assets/scene-manifest-v1.json`，需要用本地静态服务器运行。

在项目目录执行：

```powershell
python -m http.server 4173
```

然后在浏览器访问：

```text
http://127.0.0.1:4173
```

## 当前已实现

- 背景、家具、道具、角色状态图加载
- 电视互动与开关屏切换
